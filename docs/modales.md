# Modales y diálogos

Ejemplos en `src/components/modals/`:

- `createCatalogModal.jsx` — crear catálogo.
- `createPieceModal.jsx` — añadir pieza.
- `saveServiceModal.jsx` — confirmaciones al guardar.
- `shareModal.jsx` — compartir recursos.

Buenas prácticas:

- Leer mensajes y descripciones antes de confirmar acciones destructivas.
- Usar botones `Cancelar` para cerrar sin guardar.
