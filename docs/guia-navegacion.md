# Guía de navegación general

Esta sección explica la disposición general de la aplicación y cómo moverse entre pantallas.

1. Barra de navegación principal:
   - Visible en la parte superior o lateral según la pantalla.
   - Accesos directos a: `Dashboard`, `Crear Rompecabezas`, `Catálogos`, `Búsqueda`.

2. Flujo típico:
   - Inicia sesión → Dashboard → Selecciona catálago/puzzle → Accede al editor/visualizador.

3. Iconos y acciones comunes:
   - Botón `+` o `Crear`: abre modal o página para crear elemento.
   - Lupa / Search: abre la vista de búsqueda avanzada.
   - Carrito / Selecciones: abre la rejilla del carrito/representación.

4. Consejos rápidos:
   - Usa la búsqueda global para encontrar piezas o catálogos por nombre.
   - En pantallas con tablas, haz clic en encabezados para ordenar.
