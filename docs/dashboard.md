# Dashboard

Ruta: `src/app/dashboard/home/page.js`

Descripción:

El Dashboard es la vista principal tras autenticarse. Permite acceder a:

- Catálogos: lista y navegación a detalles.
- Crear Rompecabezas: iniciar flujo de creación.
- Listado de puzzles: ver, editar y ver resultados.

Elementos clave:

- Paneles resumen: muestran estado del sistema o atajos.
- Enlaces rápidos a `catalogs`, `create-puzzle`, `list-puzzles`.

Acciones frecuentes:

- Acceder a una fila para ver detalles del puzzle.
- Usar el botón `Crear` para iniciar la construcción.
