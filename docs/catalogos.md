# Catálogos

Ruta: `src/app/dashboard/catalogs/page.js`

Qué puedes hacer:

- Visualizar la lista de catálogos.
- Crear/editar catálogos usando el modal `createCatalogModal.jsx`.
- Explorar piezas asociadas a cada catálogo.

Componentes:

- `catalogComponent.jsx` — tarjeta de catálogo.
- Paginación y filtros en `AdvancedSearchFilters.jsx` cuando aplique.
