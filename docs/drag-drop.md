# Drag & Drop (arrastrar y soltar)

Componentes relacionados: `dragDrop/dragNDropList.jsx`, `dragNDropElement.jsx`, `workspace.jsx`.

Cómo funciona:

1. Selecciona una pieza y arrástrala sobre la zona de trabajo (`workspace`).
2. Suelta para colocarla; algunos elementos muestran guías de posición.
3. Usa las opciones de edición para ajustar propiedades.

Consejos:

- En pantallas con mucho contenido, utiliza zoom o filtros para encontrar elementos.
- Si algo no se mueve, revisa permisos o mensajes de validación.
