# Búsqueda avanzada

Componentes:

- `searchbar.jsx` — búsqueda global rápida.
- `AdvancedSearchFilters.jsx` — filtros por campos y criterios.

Uso recomendado:

1. Empieza con términos generales en la barra de búsqueda.
2. Refina con filtros para categoría, catálogo, o atributos.
3. Ordena resultados y utiliza paginación si hay muchas coincidencias.
