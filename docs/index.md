# Manual de Usuario — Front-nez

Bienvenido al manual de usuario para la interfaz gráfica del proyecto **Front-nez**. Este documento explica cómo navegar las pantallas principales, qué hace cada componente y pequeños flujos de trabajo para tareas frecuentes.

- Público objetivo: usuarios finales y evaluadores de la interfaz.
- Formato: guía paso a paso con capturas (si se añaden) y enlaces a secciones específicas.

Cómo usar este manual:

- Navega por el menú a la izquierda para ir a secciones concretas.
- La sección **Instalación** explica cómo servir la documentación localmente.
- En **Estructura del Proyecto** se listan los archivos relevantes para desarrolladores.

Si necesitas que añada capturas o ejemplos concretos de pantallas, indícamelo y las incorporo.
